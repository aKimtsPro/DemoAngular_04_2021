Les directives

3 types de directives :

  • component : les components
  • [attribute] : attributs permettant de modifier l'apparence d'un élément de manière dynamique
  • *structural : attributs permettant de manipuler le DOM de manière dynamique

Attribute directive

ngClass est une directive permettant d'associer une ou plusieurs classes CSS selon une condition

ngStyle permet de définir dynamique la valeurs de propriétés CSS

style.propriétéCSS permet de donner une valeur à une propriété CSS unique dynamiquement

Structural directives

*ngIf

*ngIf permet de définir la présence d'un élément selon une condition évaluée dynamiquement
