Le binding

il existe 2 types de binding

One-Way
Utilisation d'info de la classe dans le template
Two-Way
Utilisation et modification d'infos de la classe dans le template

binding One-Way

J'affiche l'age de l'utilisateur défini dans la classe : {{ age + 5 }} ans

Je donne à cet élément une classe(css) particulère défini dans la classe(ts) via binding d'atribut

binding Two-Way
